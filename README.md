WebCrawler
=======

# Preconditions 
- JDK11
- MVN 3.6

# How to run it
- From terminal launch command `mvn exec:java` (default page is http://www.dustyfeet.com)
- For specific site please provide with command `-Dexec.args="http://mysmallwebpage.com/"` with page you are interested in
